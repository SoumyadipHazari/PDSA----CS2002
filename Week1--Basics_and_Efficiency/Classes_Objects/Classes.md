# Classes and Objects

### What is an Abstract Data Type (ADT)?

it is a concept that describes -- 

- what data is stored
- what operations can be done on the data
- without telling how it is implemented inside

### What is a class?

- how data will be stored
- what functions can be used to work with this data


### What is an object?

An object is a real thing created from the class.

- Class- plan of a house
- Object - actual built house
